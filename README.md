## 🌟 Overview

**Xai – Intelligence Workspace** is a high-fidelity interactive product experience prototype designed for RacoAI's Frontend Challenge. It visually demonstrates how complex, unstructured telemetry datasets are transformed into structured vector matrices, predictive intelligence, and autonomous background AI agent executions.

---

## ✨ Features & Interactive Sections

### 1. Hero Section (Data → Intelligence Transformation)
- **3D Particle Matrix Visualizer**: Custom 3,500 point R3F vertex buffer system smoothly morphing between *Chaos Cloud*, *3D Matrix Grid*, and *Torus Neural Topology*.
- Interactive pointer tracking with raycast rotation influence and ambient lighting gradient masks.

### 2. Interactive Insight Flow (4-Stage Narrative Pipeline)
- Scroll-driven pinned section utilizing **GSAP ScrollTrigger**.
- Animated laser SVG progress path highlighting 4 core stages:
  1. *Ingest Raw Data*
  2. *Structured Intelligence*
  3. *Actionable Insights*
  4. *AI Automations*
- Interactive metric inspector cards for live stream telemetry.

### 3. Signature WOW Interaction (Interactive 3D Data Matrix)
- Interactive 3D Canvas section allowing decision-makers to manually switch topologies (*Raw Chaos*, *Vector Cluster*, *Crystal Lattice*, and *AI Topology*).
- Real-time parameter sliders for quantum turbulence factor, rotation velocity, and live HUD metrics (FPS, vector count, spatial entropy %).

### 4. Intelligence Dashboard Preview (Product UI Shell)
- Product-quality interactive workspace interface (Stripe + Linear + Vercel aesthetic).
- Multi-tab navigation with Framer Motion `layoutId` pill transitions (*Overview*, *Analytics*, *AI Agents*, *Settings*).
- Interactive AI Agent toggle switches, performance trajectory graph, and live telemetry log ticker.

## 🛠️ Technology Stack
- **Framework**: Next.js 14 (App Router, TypeScript)
- **3D Interactive Graphics**: React Three Fiber, Three.js, `@react-three/drei`
- **Animation Architecture**: GSAP (ScrollTrigger), Framer Motion
- **Design System**: Tailwind CSS, Lucide Icons, Glassmorphic Vanilla CSS

## 🚀 Running Locally

```bash
# Clone the repository
git clone https://github.com/your-username/xai-workspace.git

# Change into the workspace directory
cd xai-workspace

# Install dependencies
npm install

# Start development server
npm run dev
```

Visit `http://localhost:3000` in your browser.
